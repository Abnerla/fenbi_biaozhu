作者经常使用平板进行学习，但一直无法找到心仪且免费的屏幕标注软件，于是就开发了这一款支持跨应用全局屏幕标注的安卓APP,用户在任意应用界面均可触发标注,且标注过程不影响底层应用运行。

## 核心功能

### 1. 多种绘图工具
- **画笔**: 支持多种颜色选择,可调节粗细,实时预览效果
- **荧光笔**: 半透明高亮效果,适合标记重点内容
- **橡皮擦**: 可调节大小,精准擦除不需要的标注

### 2. 压感支持
- 根据按压力度自动调整笔画粗细(等于或小于设定值)
- 根据压感调整颜色深浅,还原真实书写手感
- 最小宽度为设定值的30%,最大为100%

### 3. 画质优化
- 抗锯齿算法,画笔边缘平滑无毛刺
- 贝塞尔曲线平滑路径,书写流畅不卡顿
- 高性能渲染,保证低延迟(<100ms响应)

### 4. 智能回溯
- 单次撤销:工具栏撤销按钮
- 连续撤销:双指双击屏幕任意位置
- 历史记录栈保存最近50步操作

### 5. 悬浮工具栏
- 右侧悬浮设计,不遮挡内容区域
- 实时预览当前画笔颜色和粗细
- 一键清空所有标注
- 快速退出标注模式
